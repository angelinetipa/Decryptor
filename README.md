# Decryptor

## Project
-	Write a Python Script that will accept a string as encrypted text and then the program will decrypt it using the following character substitute:

| letter | sign |
| -- | -- |
| a | * | 
| e | & | 
| i | # | 
| o | + |
| u | ! |

## Purpose
-	The program will decrypt the text.
